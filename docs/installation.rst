********************************************************************************
Installation
********************************************************************************

Stable
======

Stable releases are available on PyPI and can be installed with pip.

.. code-block:: bash

    pip install compas_lmgc90


Latest
======

The latest version can be installed from local source.

.. code-block:: bash

    git clone https://github.com/BlockResearchGroup/compas_lmgc90.git
    cd compas_lmgc90
    pip install -e .


Development
===========

To install `compas_lmgc90` for development, install from local source with the "dev" requirements.

.. code-block:: bash

    git clone https://github.com/BlockResearchGroup/compas_lmgc90.git
    cd compas_lmgc90
    pip install -e ".[dev]"
